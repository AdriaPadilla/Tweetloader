Command Line¶

Retrieve info from twitter of one single user¶

python3 /path/to/user_tweet_downloader/main.py –user twitter_username –id twitter_id

i.e.

python3 /users/adria/user_tweet_downloader/main.py --user adriapadilla --id 72066060

Retrieve info from twitter from multiple users¶

python3 /path/to/user_tweet_downloader/main.py –user username_1 username_2 username_n –id twitter_id_1 twitter_id_2 twitter_id_n

i.e.

python3 /users/adria/user_tweet_downloader/main.py --user adriapadilla elpais_espana --id 72066060 121183700
